Rest API support for [wakatobi](https://github.com/ardhi/wakatobi)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install wakatobi-rest-api
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```wakatobi-rest-api``` in it
