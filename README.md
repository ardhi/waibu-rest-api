Rest API support for [waibu](https://github.com/ardhi/waibu)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install waibu-rest-api
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```waibu-rest-api``` in it
